Como usar o Foto Overlay ?

Primeiramente baixe o script e transfira os arquivos na hospedagem do seu site.
Visitar Repositório Foto Overlay (.zip)

1º Passo: Habilitando o script no seu site

Após adicionar os arquivos a sua hospedagem vá até o HTML da sua página e insira os seguintes códigos em seus respectivos lugares:
  • <link href="./foto-overlay/css/foto-overlay.css" rel="stylesheet" />
  • <script src="./foto-overlay/js/foto-overlay.js"></script>
  • <script>
  • new fotoOverlay();
  • </script>

2º Passo: Chamando o script na foto

Após adicionar os códigos mostrados acima, o Foto Overlay estará habilitado e tudo o que você precisará fazer para chamá-lo é inserir as fotos desejadas utilizando o seguinte formato HTML:
  • <figure class="foto-overlay">
  • <img src="URL_DA_FOTO" alt="" />
  • </figure>
