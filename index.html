O que é o Foto Overlay

O Foto Overlay é um script que detecta quando a imagem fica menor do que a área do <figure> e exibe uma overlay utilizando a mesma imagem para que todo o espaço seja ocupado, como pode ser conferido no exemplo abaixo:
No inicio o script foi criado em jQuery, mas com o passar do tempo fui adaptando o código para que o mesmo funcione utilizando apenas Javascript puro, tornando assim, sua execução bem mais leve e compatível com os mais diversos navegadores disponíveis atualmente na web.
